更新：因为一些原因，再次切换

使用了整快硬盘切换，下次无需再部署环境了。

  1. 两台机器同时开启救援模式。
  2. 通过 Lish 进入Linode 系统后，首先修改 root 密码:passwd。
  3. 输入 root 密码。

  4. 输入以下命令启动 SSH 服务。

  5. 在旧机器上输入命令开始传输硬盘。

  6. 传输完毕，重启即可。
